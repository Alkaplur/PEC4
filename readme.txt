# Nombre del Proyecto

El presente proyecto realiza un analisis de los datos de uso de armas de fuego en Estados Unidos, y realiza la carga,
transformacion y analisis basico de los datos incluyendo la representación grafica de las proporciones relativas de
los permisos, y posesion de armas cortas y escopetas por estado.

El proyecto se ha hecho dentro del marco de la asignatura de Programación para la ciencia de datos - Aula 1 de la
Universitat Oberta de Catalunya.

No existen requisitos especificos de instalacion del proyecto, se puede usar algun IDE tipo Pycharm para la gestion de los
distintos modeulos que componen el progrma

Se permite el uso y moficinacion del codigo sin fines comerciales.

Los datos de contacto son estudiante@uoc.edu
